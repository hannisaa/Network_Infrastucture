# Network Infrastructure for Online Learning Platform

## Deskripsi proyek

Proyek ini merupakan perancangan dan simulasi infrastruktur jaringan untuk platform pembelajaran online (Online Learning Platform) menggunakan Cisco Packet Tracer.

Jaringan dirancang untuk mendukung berbagai kebutuhan operasional, seperti ruang administrasi, ruang mentor, ruang meeting, ruang live streaming, ruang server, ruang operasional, dan ruang manajer. Sistem ini mengimplementasikan layanan jaringan seperti DHCP, DNS Server, FTP Server, Web Server, serta routing antar jaringan.


## Tujuan 

* Merancang infrastruktur jaringan yang stabil dan terstruktur.
* Menghubungkan berbagai divisi dalam satu sistem jaringan.
* Mengimplementasikan pengalamatan IP statis dan DHCP.
* Mengkonfigurasi layanan DNS, FTP, dan Web Server.
* Melakukan pengujian konektivitas antar perangkat.
* Menyediakan desain jaringan yang dapat dikembangkan di masa depan.
---

## Topologi Jaringan

Jaringan dibagi menjadi beberapa segmen:

| Ruangan             | Network        |
| ------------------- | -------------- |
| Mentor Room         | 192.168.2.0/24 |
| Meeting Room        | 192.168.3.0/24 |
| Manager Room        | 192.168.4.0/24 |
| Live Streaming Room | 192.168.5.0/24 |
| Administration Room | 192.168.6.0/24 |
| Server Room         | 192.168.7.0/24 |
| Operational Room    | 192.168.8.0/24 |

---

## Teknologi yang Digunakan

* Cisco Packet Tracer
* Routing RIP
* DHCP
* DNS Server
* FTP Server
* Web Server
* Wireless Router
* Static IP Addressing
---

## Fitur Utama

* Routing antar jaringan
* DHCP untuk distribusi IP otomatis
* DNS Server untuk resolusi nama domain
* FTP Server untuk transfer file
* Web Server untuk layanan web internal
* Pengujian konektivitas menggunakan ping
* Simulasi komunikasi antar ruangan
---

## Hasil Pengujian

Pengujian dilakukan dengan:

* Ping antar perangkat pada jaringan yang berbeda.
* Pengujian koneksi ke server.
* Pengujian layanan DNS.
* Pengujian akses layanan web.
* Verifikasi distribusi IP menggunakan DHCP.
